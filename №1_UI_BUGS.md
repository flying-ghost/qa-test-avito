# Баги по дизайну (UI)

1. **Ошибка порядка фильтрации**  
   **Описание:** Неверный порядок фильтрации по страницам Главная>Животные>Персидская>Кошки.  
   **Приоритет** Р2

2. **Поиск по конкретному городу**  
   **Описание:** У пользователя указан город Москва, заголовок поиска аналогично по Москве, на ползунке указано сначала из Можайска
   **Приоритет** Р1

3. **Ошибка загрузки**  
   **Описание:** Страница загружена, но у пользователя показывается "Ошибка обновите страницу".  
   **Приоритет** Р1

4. **Ошибка фильтрации поиска**  
   **Описание:** При фильтрации по животным, у пользователя в выдаче имеется плюшевая игрушка.  
   **Приоритет** Р1

5. **Дата публикации объявления**  
   **Описание:** Не на всех карточках указана дата публикации объявления.  
   **Приоритет** Р2

6. **Некорректная кнопка фильтрации**  
   **Описание:** Для выбор возраста кошки, некорректно указана кнопка, щенок вместо котенка, что относится к другому виду животных.  
   **Приоритет** Р2

7. **Ошибка выдачи при фильтрации**  
   **Описание:** Ошибка поисковой выдачи при фильтрации цены, цена указана от 1000р, в выдаче есть объявление 500р.  
   **Приоритет** Р1

8. **Ошибка выдачи при фильтрации**  
   **Описание:** Ошибка поисковой выдачи при фильтрации цены, цена указана до 30000р, в выдаче есть объявление 40000р.  
   **Приоритет** Р1

9. **Неверная всплывающая подсказка**  
   **Описание:** При наведении на кнопку лайка, подсказка сообщает что это действие приведет к скрытию объявления из выдачи.  
   **Приоритет** Р0

10.**Грамматическая ошибка**  
   **Описание:** Кнопка Карьера в Авито написана с ошибкой.  
   **Приоритет** Р3

11.**Грамматическая ошибка**  
   **Описание:** Кнопка Уведомлять о новых объявлениях написана с ошибкой.  
   **Приоритет** Р3

12.**Некорректная фильтрация**  
   **Описание:** Некорректная фильтрация по городу, в выдаче на первом месте объявление из города Новосибирск, при выборе Москвы.  
   **Приоритет** Р1

13.**Ошибка счетчика объявлений**  
   **Описание:** В заголовке указано 19 объявлений, на кнопке и в выдаче 8 объявлений.  
   **Приоритет** Р1

14.**Ошибка верстки**  
   **Описание:** Сломана верстка карточек объявлений.  
   **Приоритет** Р2

15.**Ошибка верстки**  
   **Описание:** Сломана вертска новых сообщений в нижней части страницы, количество сообщений обрезается.  
   **Приоритет** Р2

16.**Ошибка верстки**  
   **Описание:** Сломана верстка подборки объявлений, выходит за края интерфейса.  
   **Приоритет** Р2

17.**Ошибка верстки**  
   **Описание:** Сломана вертка основной части страницы, весь интерфейс не соответствует разрешению страницы.  
   **Приоритет** Р1

18.**Ошибка фильтрации**  
   **Описание:** В окне фильтров выбран мальчик, в выдаче появляется объявление девочки.  
   **Приоритет** Р1

19.**Ошибка карточки**  
   **Описание:** По названию в выдаче появляется персидский котенок, на фото указана другая порода.  
   **Приоритет** Р1

20.**Ошибка карточки**  
   **Описание:** Появляется в выдаче, но не указана порода в названии карточки.  
   **Приоритет** Р2
