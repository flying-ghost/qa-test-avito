### ID 1
Описание: Создание объявления с отрицательным полем price возвращает 200 OK вместо 400
Приоритет: Критический
Шаги воспроизведения:
1. Сгенерировать sellerID = random(111111, 999999)
2. Отправить POST-запрос с телом: (name: Телефон, price: -98, sellerID: sellerID, statistics: (likes: 15, viewCount: 4, contacts: 45))
3. Проверить статус ответа
Фактический результат: Код 200 OK, объявление создано с price = -98
Ожидаемый результат: Код 400 Bad Request
